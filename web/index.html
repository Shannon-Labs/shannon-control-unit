Methodology

Bridging Information Theory and Control Theory.

1. The Plant

We treat the neural network training process as a dynamic system ("the plant"). The regularization coefficient $\lambda$ is the actuator, and the model's generalization state is the output.

2. The Sensor

We monitor the Information Ratio ($S$), derived from Minimum Description Length (MDL) principles. $S$ measures the balance between parameter complexity (ParamBPT) and data fit (DataBPT).

3. The Controller

A PI Controller adjusts $\lambda$ in real-time to maintain $S$ at a target setpoint ($S^*$). This feedback loop stabilizes the "learning velocity" and prevents overfitting dynamically.

The Control Law

def update_lambda(self, current_S):
    # Calculate error
    error = current_S - self.target_S
    
    # Proportional term
    P = self.Kp * error
    
    # Integral term (with anti-windup)
    self.integral += error
    self.integral = clip(self.integral, -self.limit, self.limit)
    I = self.Ki * self.integral
    
    # Control output (negative sign because dS/dLambda < 0)
    u = P + I
    
    # Update actuator
    self.current_lambda *= exp(-u)
    
    return self.current_lambda

The SCU adjusts regularization strength $\lambda$ exponentially based on the error signal from the target Information Ratio.
